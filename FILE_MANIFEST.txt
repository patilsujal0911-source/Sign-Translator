===============================================================================
CUSTOMIZATION GUIDE
===============================================================================

1. COLORS (Edit style.css :root variables)
   --bg-primary: #e8eef2           → Change background
   --accent-color: #4CAF50         → Change accent (buttons, links)
   --text-primary: #2d3436         → Change text color

2. TYPOGRAPHY (Edit style.css)
   font-family: 'Segoe UI'         → Change base font
   font-size: [xxxx]               → Adjust sizes

3. NEUMORPHISM EFFECT (Edit box-shadow values)
   --shadow-medium: [...]          → Adjust shadow depth
   --border-radius: 20px           → Adjust roundness

4. BRANDING (Edit templates/base.html)
   brand-title text                → Change app name
   navbar colors                   → Customize header

5. CONTENT (Edit templates/*.html)
   Hero text, objectives, about    → Customize messaging
   Sign catalog                    → Add real video URLs
